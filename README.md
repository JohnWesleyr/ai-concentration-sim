# AI Concentration Sim

A standalone interactive HTML simulation for the power-concentration model developed in the broader `agi-power-proof` project.

## What it shows

- physical power as a constant baseline
- lifetime labor capital collapsing as AGI advances
- public AGI capital versus private recursive capital
- UBI as support flow rather than durable power
- a tax-first, ownership-later policy heuristic
- the 50% power-share threshold where private AI owners can outweigh the rest of society

## Run it

Open `index.html` directly in a browser.
